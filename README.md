# CiteLens

Find the follow-up papers that matter most.

Paste a paper — URL, DOI, or title — and CiteLens surfaces citing work ranked by a four-signal score: **Impact**, **Network influence**, **Relevance**, and **Citation context**.

## Features

- **Three layout modes** — Focus (hero cards), Split (list + detail pane), Stream (dense table)
- **Four ranking modes** — Most Influential, Most Relevant, Recent, Reviews
- **Expandable "Why ranked here"** — per-paper formula breakdown with metric tiles
- **Filters** — year range, relevance threshold, influential-only, reviews-only
- **Timeline view** — citation arc over time
- **Dark mode** — full token-driven palette
- **Mobile viewport** — responsive layout in device frame
- **Accent colors** — Indigo, Teal, Plum, Forest, Graphite

## Usage

Open `index.html` in a browser — no build step required.

## Tech

Static HTML + CSS + React 18 (CDN) + Babel Standalone. No dependencies to install.
